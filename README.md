## About Me

I am a software engineer based in **Karachi, Pakistan**, with a strong interest in building software that is practical, scalable, and ready for real-world use.

My work combines modern full-stack development with artificial intelligence, cybersecurity, Linux systems programming, and compiler concepts. I enjoy understanding how systems work at both high and low levels—from designing responsive interfaces and APIs to working with processes, sockets, file systems, and machine-learning models.

I focus on writing clean code, solving meaningful problems, and continuously improving the quality of everything I build.
